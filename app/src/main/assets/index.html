HTML2APK ⚡

Konversi file HTML menjadi APK Android siap pakai. Gunakan GitHub API untuk build & publikasi.
1. Otentikasi GitHub (API Key)
Token memerlukan scope 'repo' & 'workflow' untuk membuat gist/repo. Simpan dengan aman.
Digunakan untuk ID aplikasi & folder (contoh: com.converter.app)
2. File HTML / Kode Sumber Web
3. Build & Konversi ke APK (via GitHub API)

Proses: kami akan membuat repository template WebView Android, mengganti asset HTML Anda, lalu memicu GitHub Actions untuk menghasilkan file APK. Setelah selesai, Anda bisa download APK langsung.

* APK akan dihasilkan oleh workflow GitHub, link download sementara berlaku 60 menit. Refresh jika perlu.
